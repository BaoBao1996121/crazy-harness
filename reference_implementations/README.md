# Reference Implementations

These files are not imported by the application.

They are the typing guide for the four hand-written modules:

- `agent_loop.py` -> `crazy_harness/core/agents/loop.py`
- `prompt_contract.py` -> `crazy_harness/core/prompts/contract.py`
- `a2a_bus.py` -> `crazy_harness/core/a2a/bus.py`
- `microcompact.py` -> `crazy_harness/core/context/microcompact.py`

Copy them manually, preferably one module at a time, and run the matching tests.
